# Troubleshooting


**Essbase/Planning User’s Password** 

If the password contains an ampersand character then application import and
deployment will fail. Ensure that the password does not contain an ampersand
character.

<br/>


**Planning Application Import or Deployment Fails with EPMLCM13000:Service currently not available error.**

This error means that the Planning application is not provisioned to the user
configured in EPMWARE.
